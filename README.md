## Run Locally

Clone the project

```bash
  git clone https://github.com/heyaarju/OnlineKYC.git
```

Go to the project directory

```bash
  cd OnlineKYC
```

Install dependencies

```bash
  pip3 install -r r.txt
```

Start the server

```bash
  python3 app.py runserver
```
